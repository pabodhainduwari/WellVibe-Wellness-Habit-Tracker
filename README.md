# 🌱 WellVibe – Wellness & Habit Tracker

WellVibe is a mobile application designed to help users build healthy habits, track their wellness journey, and improve their daily lifestyle through habit management, mood tracking, hydration monitoring, and personalized reminders.

---

## 📱 Features

### ✅ Habit Tracking
- Create and manage daily habits
- Track completion progress
- View habit streaks and achievements
- Interactive habit dashboard

### 😊 Mood Monitoring
- Record daily moods
- Mood journal functionality
- Mood trend visualization
- Personal wellness insights

### 💧 Hydration Tracking
- Daily water intake tracking
- Custom hydration goals
- Water consumption reminders
- Progress monitoring dashboard

### 🔔 Smart Reminders
- Habit reminders
- Hydration notifications
- Wellness alerts
- Personalized scheduling

### 📊 Analytics & Progress
- Weekly summaries
- Progress reports
- Achievement tracking
- Habit performance insights
